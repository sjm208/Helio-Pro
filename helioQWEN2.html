Photometric Stereo &
RTI for Cultural Heritage

A high-performance desktop application for quantitative surface imaging, 3D mesh analysis, and archival metrology. Open source. MIT License.

Get the Code Read the Docs

40+ Engines

Advanced photometric stereo processing, spectral analysis, and real-time relighting capabilities.

ISO Metrology

Quantitative depth analysis compliant with ISO 25178 and ISO 4287 standards.

3D Export

Interactive viewer and export pipelines for OBJ, GLB, PLY, STL, and E57 formats.

AI Integration

Built-in Claude API support for surface analysis and automated condition reporting.

Interface Showcase

From the dome calibrator to the archival export pipeline.

Virtual Dome Calibrator
10-tool calibration system for rigs without a chrome sphere. Supports `.lp` file import.
Interactive 3D Mesh
OpenGL-based viewer reconstructing depth data into high-fidelity 3D geometry.
Clay Render Mode
Toggle to visualize surface relief and texture without the distraction of color data.
Export Pipeline
Launch external viewers and process meshes directly within the application.
Archive & Metadata
Generate CIDOC-CRM, IIIF 3.0 manifests, and Heritage Archive ZIPs.
Metrology & QC
Real-time adjustment of roughness parameters and surface normal quality control.

Ready to use?

Get up and running in seconds. Python 3.10+ required.

# Clone the repository

$ git clone https://github.com/sjm208/Helio-Pro.git
$ cd Helio-Pro

# Install dependencies

$ pip install -r requirements.txt

# Launch the application

$ python Helio_Pro_V1.py
